# Haiku Documentation


## Haiku Audio Recording

### 1. Poem Selection  
**The light of a candle**  
Is transferred to another candle —  
spring twilight.

#### 2. Process  
1. Create aiff file using Terminal 
   `
   say " The light of a candle is transferred to another candle - spring twilight." -o Haiku.aiff
   
   `

2. Move the aiff file to itp/Haiku Folder
	`
	mv Haiku.aiff /Users/anda/Documents/GitHub/itp/Haiku 
	
	`
